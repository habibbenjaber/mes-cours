1. Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation d'un document écrit en HTML. Il permet de contrôler l'apparence des éléments d'une page web, notamment :

  • Couleurs et fonds : Couleur du texte, couleur de fond, images de fond
  • Typographie : Police, taille, graisse, espacement
  • Disposition : Positionnement, mise en page, grilles
  • Animations : Transitions, transformations, animations

2. Où écrire le CSS ?

Il existe trois méthodes principales pour intégrer du CSS dans vos pages web :

/* 1. CSS Interne - Dans la balise <style> */
<style>
  body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
  }
</style>

/* 2. CSS Externe - Fichier séparé */
<link rel="stylesheet" href="styles.css">

/* 3. CSS Inline - Directement dans l'élément */
<h1 style="color: blue; font-size: 24px;">Titre</h1>

✅ Recommandé

  • CSS Externe (fichier .css)
  • CSS Interne (balise <style>)

❌ À éviter

  • CSS Inline (attribut style)
  • Mélanger plusieurs méthodes

3. Structure d'une règle CSS

Une règle CSS est composée d'un sélecteur et d'une déclaration :

/* Structure d'une règle CSS */
sélecteur {
  propriété: valeur;
  propriété: valeur;
}

/* Exemple */
h1 {
  color: blue;
  font-size: 2rem;
  text-align: center;
}

4. Les sélecteurs CSS simples

Les sélecteurs permettent de cibler les éléments HTML à styliser :

Sélecteur d'élément (Type)

Syntaxe : nom_de_l_element

Exemple : p { color: red; }

Cible : Tous les éléments <p>

Sélecteur de classe

Syntaxe : .nom_de_la_classe

Exemple : .important { font-weight: bold; }

Cible : Tous les éléments avec class="important"

Sélecteur d'ID

Syntaxe : #nom_de_l_id

Exemple : #header { background: gray; }

Cible : L'élément avec id="header"

Sélecteur universel

Syntaxe : *

Exemple : * { margin: 0; padding: 0; }

Cible : Tous les éléments

Sélecteur d'attribut

Syntaxe : [attribut] ou [attribut=valeur]

Exemple : input[type="text"] { border: 1px solid blue; }

Cible : Éléments avec l'attribut spécifié

5. La cascade et l'héritage

Le CSS suit des règles de priorité pour déterminer quels styles s'appliquent :

/* Ordre de priorité (du plus faible au plus fort) */
1. Styles du navigateur (user agent)
2. Styles externes et internes
3. Styles inline (attribut style)
4. Styles avec !important

/* Exemple de cascade */
p { color: black; } /* Règle externe */
<style>
  p { color: blue; } /* Règle interne - prioritaire */
</style>
<p style="color: red;">Texte rouge</p> /* Inline - le plus prioritaire */

⚠️ Règles importantes :

  • Spécificité : Plus un sélecteur est spécifique, plus il est prioritaire
  • Ordre : Les règles définies plus tard écrasent les précédentes
  • Héritage : Certaines propriétés sont héritées par les éléments enfants
  • !important : À utiliser avec parcimonie, casse la cascade normale

Exemple 2: Application de différents sélecteurs

Observez comment différents sélecteurs affectent les éléments :

HTML de test :

<div id="main">
  <p>Paragraphe normal</p>
  <p class="special">Paragraphe spécial</p>
  <p class="special important">Paragraphe important</p>
  <div>
    <span>Span dans un div</span>
  </div>
</div>

CSS appliqué :

/* Sélecteur d'élément */
p { color: #333; }

/* Sélecteur de classe */
.special { font-weight: bold; }

/* Sélecteur multiple */
.special.important { color: red; }

/* Sélecteur descendant */
div span { background-color: yellow; }
