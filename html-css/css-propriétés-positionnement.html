1. La propriété position

La propriété position définit comment un élément est positionné dans le document :

static (par défaut)

L'élément suit le flux normal du document. Il n'est pas affecté par les propriétés top, bottom, left, right ou z-index.

position: static;
Static
Static

relative

L'élément est positionné relativement à sa position normale dans le flux du document. L'espace originellement occupé par l'élément est conservé.

position: relative;
top: 20px;
left: 30px;
Static
Relative
Static

Note : La zone en pointillés bleus montre la position normale de l'élément. L'élément vert est décalé de 20px vers le bas et 30px vers la droite par rapport à sa position d'origine.

absolute

Positionné relativement au premier parent positionné (non static). L'élément est retiré du flux normal.

position: absolute;
top: 50px;
right: 20px;
Static
Absolute

fixed

Positionné relativement à la fenêtre du navigateur. L'élément reste fixe pendant le défilement.

position: fixed;
bottom: 20px;
right: 20px;
Static
Fixed

sticky

Se comporte comme relative jusqu'à un certain point de défilement, puis comme fixed.

position: sticky;
top: 0;
Sticky

2. Les propriétés de positionnement

Une fois la position définie, on utilise ces propriétés pour le placement :

top, right, bottom, left

Décalage par rapport aux bords du conteneur.

top: 10px;
right: 20px;
bottom: 30px;
left: 40px;

z-index

Contrôle l'ordre d'empilement des éléments.

z-index: 10;

3. z-index et l'empilement

La propriété z-index contrôle l'ordre d'empilement des éléments positionnés :

z-index: 1
z-index: 2
z-index: 3
/* Élément en arrière-plan */
.element1 {
  position: absolute;
  z-index: 1;
}

/* Élément au premier plan */
.element2 {
  position: absolute;
  z-index: 10;
}

5. Flexbox (Disposition flexible)

Flexbox est un modèle de mise en page unidimensionnel :

Item 1
Item 2
Item 3
/* Conteneur flex */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* Éléments flex */
.item {
  flex: 1;
}

📝 Règles importantes :

  • position: relative conserve l'espace originel de l'élément dans le flux
  • position: absolute se positionne par rapport au premier parent avec position différente de static
  • position: fixed est toujours relatif à la fenêtre du navigateur
  • z-index ne fonctionne qu'avec les éléments positionnés (non static)
  • Flexbox et Grid sont les méthodes modernes pour la mise en page
  • Utilisez float principalement pour l'habillage du texte autour des images

💡 Cas d'utilisation courants :

  • position: relative : Décalages mineurs, conteneurs pour éléments absolus
  • position: absolute : Tooltips, menus déroulants, badges, overlays
  • position: fixed : En-têtes fixes, boutons d'action flottants
  • position: sticky : En-têtes de tableaux, menus de navigation
  • z-index : Modales, menus déroulants, éléments superposés
  • Flexbox : Mises en page unidimensionnelles, navigation
  • Grid : Mises en page complexes, grilles de produits
