1. Qu'est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il définit la structure et le contenu d'une page web :

  • Structure : Organisation hiérarchique du contenu
  • Sémantique : Signification des différents éléments
  • Contenu : Texte, images, liens, etc.
  • Accessibilité : Structure compréhensible par les technologies d'assistance

2. Les Balises HTML de Base

Les balises HTML sont des mots-clés entourés de chevrons (< et >) qui indiquent au navigateur comment afficher le contenu.

/* Structure de base d'une balise */
<nom-balise>Contenu</nom-balise>

Exemples de balises courantes

<h1> à <h6> : Titres de différents niveaux

<p> : Paragraphe

<a> : Lien hypertexte

<img> : Image

<ul> et <li> : Liste non ordonnée et éléments de liste

<div> : Conteneur générique

⚠️ Règles importantes :

  • La plupart des balises ont une balise ouvrante et une balise fermante
  • Certaines balises comme <img> et <br> sont auto-fermantes
  • L'imbrication doit être correcte - fermez les balises dans l'ordre inverse de leur ouverture

3. Les Attributs HTML

Les attributs fournissent des informations supplémentaires sur les éléments. Ils sont toujours spécifiés dans la balise ouvrante.

/* Structure d'un attribut */
<balise nom-attribut="valeur">

Attributs courants

id : Identifiant unique pour un élément

class : Classe pour le style CSS

src : Source pour les images, scripts, etc.

href : Lien pour les balises <a>

alt : Texte alternatif pour les images

style : Styles CSS inline

4. Les Éléments HTML

Un élément HTML est constitué d'une balise ouvrante, d'un contenu et d'une balise fermante.

/* Structure d'un élément */
<balise attribut="valeur">Contenu</balise>

Exemples d'éléments

<p>Ceci est un paragraphe.</p>
<a href="https://example.com">Visitez Example</a>
<img src="image.jpg" alt="Description">

Les Types de Balises HTML

En HTML, les balises peuvent être classées selon deux critères : leur comportement d'affichage et leur structure syntaxique.

Classification par Comportement d'Affichage

Balises de Bloc (Block)

Les balises de bloc créent un nouveau bloc et occupent toute la largeur disponible. Elles commencent toujours sur une nouvelle ligne.

Élément Bloc 1
Élément Bloc 2
Élément Bloc 3
Exemples de balises bloc :
  • <div> - Conteneur générique
  • <p> - Paragraphe
  • <h1> à <h6> - Titres
  • <ul>, <ol> - Listes
  • <li> - Élément de liste
  • <section> - Section

Balises en Ligne (Inline)

Les balises en ligne s'insèrent dans le flux du texte et n'occupent que l'espace nécessaire. Elles ne créent pas de nouvelle ligne.

Ceci est un texte avec élément inline qui continue sur la même ligne sans interruption.
Exemples de balises en ligne :
  • <span> - Conteneur générique inline
  • <a> - Lien
  • <strong>, <em> - Mise en forme
  • <b>, <i> - Style
  • <img> - Image
  • <code> - Code informatique
Classification par Structure Syntaxique

Balises en Paire

Les balises en paire nécessitent une balise ouvrante et une balise fermante. Elles contiennent du texte ou d'autres éléments.

<balise-ouvrante>Contenu</balise-fermante>
Exemples de balises en paire :
  • <p>...</p> - Paragraphe
  • <div>...</div> - Division
  • <h1>...</h1> - Titre
  • <a>...</a> - Lien
  • <span>...</span> - Span
  • <strong>...</strong> - Texte important

Balises Orphelines (Auto-fermantes)

Les balises orphelines n'ont pas de contenu et ne nécessitent pas de balise fermante. Elles sont auto-fermantes.

<balise attribut="valeur">
Exemples de balises orphelines :
  • <img> - Image
  • <br> - Saut de ligne
  • <hr> - Ligne horizontale
  • <input> - Champ de formulaire
  • <meta> - Métadonnées
  • <link> - Liaison externe

Les Balises de Mise en Forme

HTML propose plusieurs balises pour mettre en forme le texte sans avoir besoin de CSS.

Balise Description Utilisation
<b> Texte en gras (style uniquement) Mettre en évidence visuellement
<strong> Texte fort (sémantique + style) Indiquer l'importance du contenu
<i> Texte en italique (style uniquement) Termes techniques, pensées
<em> Texte emphatisé (sémantique + style) Mettre l'accent sur un mot
<u> Texte souligné Mots-clés, noms propres
<s> Texte barré Information obsolète
