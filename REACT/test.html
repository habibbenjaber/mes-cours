Atelier CSS - Propriétés de mise en forme

Introduction aux propriétés CSS

Cet atelier couvre les propriétés fondamentales de mise en forme en CSS, y compris les propriétés de texte, d'arrière-plan, de bordures, de dimensions et les unités de mesure.

Propriétés de mise en forme du texte

Les propriétés de texte permettent de contrôler l'apparence du texte dans vos pages web.

Propriété Description Valeurs courantes
color Définit la couleur du texte nom (red), hex (#ff0000), rgb(r,g,b)
font-family Définit la police de caractères Arial, "Times New Roman", sans-serif
font-size Définit la taille de la police px, em, rem, %
font-weight Définit l'épaisseur de la police normal, bold, 100-900
text-align Aligne le texte horizontalement left, right, center, justify
text-decoration Ajoute des décorations au texte none, underline, overline, line-through
line-height Définit l'espacement entre les lignes normal, nombre, px, %

Exemples de propriétés de texte

Ce texte est en rouge et en gras

Ce texte utilise une police monospace et est souligné

Ce texte est centré et a une taille de 24px

Ce texte a un interligne de 2 (double interligne)

/* Exemple d'utilisation des propriétés de texte */ p { color: #333; font-family: Arial, sans-serif; font-size: 16px; text-align: justify; line-height: 1.5; }

Propriétés d'arrière-plan

Les propriétés d'arrière-plan permettent de styliser l'arrière-plan des éléments.

Propriété Description Valeurs courantes
background-color Définit la couleur d'arrière-plan nom, hex, rgb, rgba, transparent
background-image Définit une image d'arrière-plan url('chemin/vers/image.jpg')
background-repeat Définit comment l'image se répète repeat, no-repeat, repeat-x, repeat-y
background-position Définit la position de l'image top, bottom, left, right, center, px, %
background-size Définit la taille de l'image auto, cover, contain, px, %

Exemples de propriétés d'arrière-plan

Arrière-plan avec couleur unie

Arrière-plan avec dégradé

/* Exemple d'utilisation des propriétés d'arrière-plan */ .element { background-color: #f0f0f0; background-image: url('texture.png'); background-repeat: no-repeat; background-position: center; background-size: cover; } /* Version raccourcie (shorthand) */ .element { background: #f0f0f0 url('texture.png') no-repeat center / cover; }

Propriétés de bordures

Les propriétés de bordures permettent de styliser les bordures des éléments.

Propriété Description Valeurs courantes
border-width Définit l'épaisseur de la bordure px, thin, medium, thick
border-style Définit le style de la bordure none, solid, dotted, dashed, double
border-color Définit la couleur de la bordure nom, hex, rgb, rgba
border-radius Définit le rayon des coins arrondis px, %

Exemples de propriétés de bordures

Bordure solide rouge de 2px
Bordure pointillée bleue de 3px
Bordure pointillée verte avec coins arrondis
/* Exemple d'utilisation des propriétés de bordures */ .element { border-width: 2px; border-style: solid; border-color: #333; border-radius: 5px; } /* Version raccourcie (shorthand) */ .element { border: 2px solid #333; border-radius: 5px; } /* Bordures individuelles */ .element { border-top: 1px solid red; border-right: 2px dashed blue; border-bottom: 1px solid green; border-left: 2px dotted purple; }

Propriétés de dimensions

Les propriétés de dimensions permettent de contrôler la taille et l'espacement des éléments.

Propriété Description Valeurs courantes
width Définit la largeur d'un élément px, %, auto
height Définit la hauteur d'un élément px, %, auto
margin Définit l'espace extérieur autour d'un élément px, %, auto
padding Définit l'espace intérieur d'un élément px, %

Exemples de propriétés de dimensions

Largeur: 200px, Hauteur: 100px
Largeur: 50%, Hauteur: 80px, Marge: 20px haut/bas
Largeur: 300px, Hauteur: 120px, Padding: 20px, Marge: centré
/* Exemple d'utilisation des propriétés de dimensions */ .element { width: 300px; height: 200px; margin: 20px; padding: 15px; } /* Marges et padding individuels */ .element { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; padding-top: 15px; padding-right: 25px; padding-bottom: 15px; padding-left: 25px; }

Unités de mesure en CSS

CSS propose différentes unités de mesure pour définir les dimensions et espacements.

Unités absolues

Les unités absolues ont une taille fixe qui ne change pas selon le contexte.

Unité Description
px Pixels - 1px = 1/96ème de pouce
pt Points - 1pt = 1/72ème de pouce
cm Centimètres
mm Millimètres
in Pouces - 1in = 2.54cm = 96px

Unités relatives

Les unités relatives sont calculées par rapport à une autre valeur (taille de police parent, taille de fenêtre, etc.).

Unité Description
% Pourcentage - relatif à la taille de l'élément parent
em Relatif à la taille de police de l'élément parent
rem Relatif à la taille de police de l'élément racine (html)
vw 1% de la largeur de la fenêtre (viewport width)
vh 1% de la hauteur de la fenêtre (viewport height)

Exemples d'unités de mesure

Largeur: 50% (relative au parent)
Largeur: 300px (absolue)
Largeur: 50vw (50% de la largeur de la fenêtre)
Taille: 1.5em (30px), Largeur: 20em (400px)
/* Exemples d'utilisation des unités */ .element-absolu { width: 300px; height: 200px; font-size: 16pt; } .element-relatif { width: 50%; height: 80vh; font-size: 1.2rem; margin: 2em; }

Méthode Shorthand (raccourcis)

La méthode shorthand permet de définir plusieurs propriétés CSS en une seule déclaration.

Propriétés avec raccourcis courants

Margin et Padding

/* Syntaxe complète */ .element { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; } /* Syntaxe raccourcie */ .element { margin: 10px 20px 10px 20px; /* haut droite bas gauche */ } /* Syntaxe encore plus courte */ .element { margin: 10px 20px; /* haut/bas droite/gauche */ } /* Une seule valeur pour les 4 côtés */ .element { margin: 15px; /* haut droite bas gauche = 15px */ }

Border

/* Syntaxe complète */ .element { border-width: 2px; border-style: solid; border-color: #333; } /* Syntaxe raccourcie */ .element { border: 2px solid #333; }

Background

/* Syntaxe complète */ .element { background-color: #f0f0f0; background-image: url('image.jpg'); background-repeat: no-repeat; background-position: center; background-size: cover; } /* Syntaxe raccourcie */ .element { background: #f0f0f0 url('image.jpg') no-repeat center / cover; }

Font

/* Syntaxe complète */ .element { font-style: italic; font-variant: normal; font-weight: bold; font-size: 16px; line-height: 1.5; font-family: Arial, sans-serif; } /* Syntaxe raccourcie */ .element { font: italic normal bold 16px/1.5 Arial, sans-serif; }
