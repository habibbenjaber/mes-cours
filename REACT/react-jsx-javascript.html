1. Qu'est-ce que le JSX ?

Le JSX (JavaScript XML) est une extension de syntaxe pour JavaScript qui permet d'écrire du code ressemblant à du HTML directement dans vos fichiers JavaScript. C'est la pierre angulaire de React (le concept central et essentiel).

Définition Technique

Le JSX est un sucre syntaxique (extension) pour React.createElement(). Il est transpilé ("traduit")en JavaScript pur avant d'être exécuté.

Avantages

Syntaxe familière (HTML-like), meilleure lisibilité, détection d'erreurs à la transpilation, et outils de développement améliorés.

📝 JSX

const element = <h1 className="title">Hello World!</h1>;

⚙️ JavaScript Compilé

const element = React.createElement(
  'h1',
  { className: 'title' },
  'Hello World!'
);

2. Règles Fondamentales du JSX

2.1 Un seul élément parent

✅ Correct

return (
  <div>
    <h1>Titre</h1>
    <p>Paragraphe</p>
  </div>
);

❌ Incorrect

return (
  <h1>Titre</h1>
  <p>Paragraphe</p>
);

2.2 Toutes les balises doivent être fermées

✅ Correct

<img src="image.jpg" alt="Description" />
<input type="text" />
<br />

❌ Incorrect

<img src="image.jpg" alt="Description">
<input type="text">
<br>

2.3 className au lieu de class

✅ Correct

<div className="container">...</div>

❌ Incorrect

<div class="container">...</div>

2.4 htmlFor au lieu de for

✅ Correct

<label htmlFor="email">Email</label>

❌ Incorrect

<label for="email">Email</label>

3. Intégration de JavaScript dans le JSX

Le JSX permet d'intégrer directement des expressions JavaScript en utilisant les accolades {}.

3.1 Variables et expressions simples

function Welcome() {
  const name = "Salem";
  const age = 25;
  const isStudent = true;

  return (
    <div>
      <!-- Affichage de variables -->
      <h1>Bonjour, {name} !</h1>
      
      <!-- Expressions mathématiques -->
      <p>Vous avez {age} ans -       
      <!-- Expressions booléennes -->
      <p>Statut: {isStudent ? "Étudiant" : "Professionnel"}</p>
    </div>
  );
}

3.2 Appels de fonctions

function formatDate(date) {
  return new Date(date).toLocaleDateString('fr-FR');
}

function DateDisplay() {
  const currentDate = "2024-01-15";
  
  return (
    <div>
      <p>Date formatée: {formatDate(currentDate)}</p>
      <p>Timestamp actuel: {Date.now()}</p>
    </div>
  );
}

3.3 Tableaux et .map()

function List() {
  const items = ['Pommes', 'Bananes', 'Oranges'];
  const numbers = [1, 2, 3, 4, 5];

  return (
    <div>
      <!-- Transformation de tableau avec .map() -->
      <ul>
        {items.map((item, index) => (
          <li key={index}>{item}</li>
        ))}
      </ul>
      
      <!-- Calculs sur tableaux -->
      <p>Somme: {numbers.reduce((a, b) => a + b, 0)}</p>
      <p>Moyenne: {numbers.reduce((a, b) => a + b, 0) / numbers.length}</p>
    </div>
  );
}

4. Attributs Dynamiques

Les attributs JSX peuvent également contenir des expressions JavaScript.

4.1 Styles dynamiques

function StyledBox() {
  const isActive = true;
  const boxSize = 200;

  return (
    <div
      style={{
        width: boxSize + 'px',
        height: boxSize + 'px',
        backgroundColor: isActive ? '#61dafb' : '#ccc',
        border: isActive ? '2px solid #21a1c4' : '1px solid #999'
      }}
    >
      Boîte {isActive ? "active" : "inactive"}
    </div>
  );
}

4.2 Classes conditionnelles

function Button({ isPrimary, isDisabled }) {
  const buttonClass = 'btn ' + (
    isPrimary ? 'btn-primary ' : 'btn-secondary '
  ) + (isDisabled ? 'disabled' : '');

  return (
    <button className={buttonClass} disabled={isDisabled}>
      Cliquez-moi
    </button>
  );
}

4.3 Attributs dynamiques avancés

function DynamicImage() {
  const imageConfig = {
    src: "https://example.com/image.jpg",
    alt: "Une image descriptive",
    width: 300,
    height: 200
  };

  return (
    <img
      src={imageConfig.src}
      alt={imageConfig.alt}
      width={imageConfig.width}
      height={imageConfig.height}
      style={{
        border: '2px solid ' + (imageConfig.width > 400 ? 'red' : 'blue')
      }}
    />
  );
}

5. Pièges Courants et Bonnes Pratiques

5.1 Ne pas utiliser d'instructions dans les accolades

✅ Expressions seulement

{variable}
{array.map()}
{condition ? a : b}
{functionCall()}

❌ Instructions interdites

{if (condition) {}}
{for (let i=0; i<10; i++) {}}
{switch (value) {}}
{variable = value}

5.2 Toujours fournir une key avec .map()

✅ Avec key

{items.map((item, index) => (
  <li key={item.id}>...</li>
))}

❌ Sans key

{items.map(item => (
  <li>...</li>
))}

💡 Règle d'Or : Les accolades {} ne contiennent que des expressions

Une expression retourne toujours une valeur. Une instruction effectue une action mais ne retourne pas de valeur.

Démonstration 3 : Attributs Dynamiques

Objectif Pédagogique

Apprendre à utiliser des expressions JavaScript pour définir dynamiquement les attributs des éléments.

function DynamicAttributes() {
  const [theme, setTheme] = React.useState('light');
  const [size, setSize] = React.useState(100);

  const themeStyles = {
    light: {
      background: '#ffffff',
      color: '#333333',
      border: '2px solid #61dafb'
    },
    dark: {
      background: '#2c3e50',
      color: '#ecf0f1',
      border: '2px solid #34495e'
    }
  };

  return (
    <div>
      <div className="controls">
        <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
          Changer le thème
        </button>
        <input
          type="range"
          min="50"
          max="200"
          value={size}
          onChange={(e) => setSize(parseInt(e.target.value))}
        />
      </div>
      
      <div
        style={{{
          ...themeStyles[theme],
          width: size + 'px',
          height: size + 'px',
          padding: '20px',
          margin: '20px auto',
          textAlign: 'center',
          borderRadius: '8px'
        }}}
      >
        <h3>{theme === 'light' ? '☀️ Mode Clair' : '🌙 Mode Sombre'}</h3>
        <p>Taille: {size}px</p>
        <p>Aire: {size * size}px²</p>
      </div>
    </div>
  );
}

Démonstration 4 : Expressions Complexes

Objectif Pédagogique

Explorer les limites des expressions JavaScript dans le JSX avec des cas d'usage avancés.

function ComplexExpressions() {
  const products = [
    { id: 1, name: "Laptop", price: 999, category: "electronics" },
    { id: 2, name: "Book", price: 25, category: "education" },
    { id: 3, name: "Headphones", price: 150, category: "electronics" },
    { id: 4, name: "Notebook", price: 5, category: "education" }
  ];

  const [filter, setFilter] = React.useState('all');

  const filteredProducts = products.filter(product =>
    filter === 'all' || product.category === filter
  );

  const totalValue = filteredProducts.reduce(
    (sum, product) => sum + product.price, 0
  );

  const averagePrice = filteredProducts.length > 0
    ? totalValue / filteredProducts.length
    : 0;

  return (
    <div>
      <div className="filters">
        {['all', 'electronics', 'education'].map(category => (
          <button
            key={category}
            onClick={() => setFilter(category)}
            style={{
              background: filter === category ? '#61dafb' : '#f8f9fa'
            }}
          >
            {category === 'all' ? 'Tous' : category}
          </button>
        ))}
      </div>
      
      <div className="stats">
        <p>Nombre de produits: {filteredProducts.length}</p>
        <p>Valeur totale: {totalValue} dt </p>
        <p>Prix moyen: {averagePrice.toFixed(2)} dt </p>
      </div>
      
      <div className="products">
        {filteredProducts.map(product => (
          <div key={product.id} className="product-card">
            <h4>{product.name}</h4>
            <p>Prix: {product.price} dt</p>
            <p>Catégorie: {product.category}</p>
            <p style={{
              color: product.price > averagePrice ? '#dc3545' : '#28a745',
              fontWeight: 'bold'
            }}>
              {product.price > averagePrice ? 'Au-dessus' : 'En-dessous'} de la moyenne
            </p>
          </div>
        ))}
      </div>
    </div>
  );
}

🎓 Exercice Final - Créer un Composant Dynamique

Scénario : Développez un composant de profil utilisateur avec des données dynamiques.

Fonctionnalités requises :

  • Afficher le nom, l'âge et l'email de l'utilisateur
  • Calculer et afficher l'année de naissance
  • Afficher une liste de compétences avec .map()
  • Utiliser des styles conditionnels basés sur l'âge
  • Ajouter un indicateur de statut en ligne/hors ligne
function UserProfile() {
  const user = {
    name: "Jean Dupont",
    age: 32,
    email: "jean.dupont@email.com",
    isOnline: true,
    skills: ["JavaScript", "React", "Node.js", "CSS"]
  };

  const currentYear = new Date().getFullYear();
  const birthYear = currentYear - user.age;

  return (
    <div style={{
      border: '2px solid #61dafb',
      borderRadius: '10px',
      padding: '20px',
      maxWidth: '400px',
      margin: '20px auto',
      background: user.age > 30 ? '#f8f9fa' : '#e8f4fc'
    }}>
      <h2>{user.name}</h2>
      <p>Âge: {user.age} ans</p>
      <p>Né en: {birthYear}</p>
      <p>Email: {user.email}</p>
      <p>
        Statut: {user.isOnline ? '🟢 En ligne' : '🔴 Hors ligne'}
      </p>
      <div>
        <h3>Compétences ({user.skills.length})</h3>
        <ul>
          {user.skills.map((skill, index) => (
            <li key={index}>{skill}</li>
          ))}
        </ul>
      </div>
    </div>
  );
}
